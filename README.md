# Codexa - Pesquisa de Satisfação

Página estática para enviar aos clientes após a entrega ou durante o fechamento do projeto.

## O que contém

- Formulário de satisfação do cliente
- Nota NPS de 0 a 10
- Perguntas sobre atendimento, comunicação, prazo, organização e resultado final
- Campos abertos para elogios, melhorias e comentário final
- Autorização para uso do feedback como depoimento
- Envio das respostas direto para o WhatsApp da Codexa
- Botão para copiar as respostas
- Layout responsivo usando identidade visual da Codexa

## Como alterar o WhatsApp

No arquivo `script.js`, altere:

```js
const WHATSAPP_NUMBER = '5548988512030';
```

Use o número com código do país e DDD, sem `+`, espaços ou símbolos.

## Deploy

Pode publicar direto na Vercel, Netlify ou qualquer hospedagem estática.
